# Towers of Hanoi

A demo of the solution to this tradional puzzle.

**History & Details: [https://en.wikipedia.org/wiki/Tower_of_Hanoi](https://en.wikipedia.org/wiki/Tower_of_Hanoi)**

**Live Demo:** [http://www.darrought66.com/](http://www.darrought66.com/)

## Tech Stack

- **Angular** 21 (standalone components, signals / `model()` where used)
- **TypeScript** 5.9
- **Unit tests:** `ng test` (Vitest via Angular’s test builder—see `angular.json`)
- Node v25.9.0

## Getting Started

```bash
npm install
npm start
```

Then open the URL shown in the terminal (usually `http://localhost:4200/`).

Other scripts:


| Command         | Description                     |
| --------------- | ------------------------------- |
| `npm run build` | Production build output         |
| `npm run watch` | Development build in watch mode |
| `npm test`      | Unit tests                      |
